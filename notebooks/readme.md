# Notebooks

Esta pasta contém os notebooks utilizados na análise exploratória dos dados (EDA).

## Conteúdo
- Entendimento do problema de negócio
- Definição da variável target
- Análise das variáveis operacionais (logística, atendimento, pedidos)

## Objetivo
Explorar os dados e identificar os principais fatores associados à satisfação e insatisfação dos clientes, com foco em geração de insights de negócio.
