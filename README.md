# AI_SCIENTIST_TECH-CHALLENGE_1
Exploratory analysis of customer satisfaction (NPS) drivers in e-commerce with focus on operational insights and business understanding

## Objetivo
Este projeto tem como objetivo analisar os principais fatores que impactam a satisfação do cliente (NPS) em um e-commerce, com foco em entendimento de negócio e geração de insights acionáveis.

## Problema de negócio
Quais fatores operacionais mais influenciam a satisfação do cliente e como podemos mitigar o risco de clientes se tornarem detratores

## Escopo do projeto
Este trabalho contempla:
- Entendimento do problema de negócio
- Definição da variável target
- Análise exploratória dos dados (EDA)

## Abordagem
- Exploração dos dados com foco em negócio
- Análise de variáveis operacionais (logística, atendimento, pedido)
- Identificação de padrões de insatisfação

## Principais insights
- Atrasos na entrega estão associados a maior insatisfação
- Clientes com mais contatos com suporte tendem a ser detratores
- Tempo de resolução elevado impacta negativamente a experiência
- Detratores apresentam menor taxa de recompra

## Recomendações
- Reduzir atrasos logísticos e melhorar consistência das entregas
- Monitorar sinais de fricção na jornada (reclamações, contatos)
- Atuar na causa raiz dos problemas operacionais

## Estrutura do projeto
- `data/`: base de dados
- `notebooks/`: análise exploratória
- `reports/`: apresentação executiva e vídeo

## Como executar

2. Instalar dependências
3. Executar notebooks
